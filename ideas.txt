## Este repositorio contiene la configuración base para poder ejecutar Visual Studio Code en un Docker Container.
## El contenedor se ejecuta sobre linux, con python 3.10.
## Se instalarán automáticamente las siguientes librerías.

- pydoc-markdown
- pyyaml
- colored
- ipykernel
- numpy
- pandas
- matplotlib
- torch torchvision torchaudio 
- python-dateutil
- pdf2image
- pdfminer
- pdfminer.six
- openai
- scikit-learn
- rich
- tqdm
- futures
- pyautogen
- nbconvert
- pandoc
- ipypublish
